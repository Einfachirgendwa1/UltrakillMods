### Compilation from Source

#### Variant 1.

Run the following command to create MessingAround.zip:

```commandline
python build_zip.py
```

You can then install the created MessingAround.zip by opening r2modman and navigate to Settings > Import local mod. Then
just select MessingAround.zip to install.

#### Variant 2.

Create a r2modman profile called dev and manually install all dependencies. Then run:

```commandline
python dev.py
```

The mod will not show up in r2modman if installed via Variant 2.
